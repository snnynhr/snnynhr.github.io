Projects

MuSc

A web-based application which is able to split media across multiple devices, so that multiple laptops or phones could be used as a combined display.

Built with:

Node.js, JavaScript, ffmpeg

ListenUP

An application which listens when someone calls your name and subsequently lowers the volume of your computer and sends a notification to alert you. In this way, you can seamlessly listen to music and be aware of your surroundings. Won "Best Hack People Will Use Everyday" at PennAppsX.

Built with:

Google's Speech to Text, NLP, PyAudio

ChromeTools

A Chrome Extension which provides quick access to essential Chrome settings, and provides an interface for customized control of the browser's stored data, such as browsing history, cookies, tabs, and more.

Built with:

JavaScript, CSS, HTML, Chrome Extension

YouTubeTracker

YouTubeTracker is a tool that manages YouTube history and displays recently watched videos, and additionally has the feature to convert and download the video as an MP3 file.

Built with:

JavaScript, CSS, HTML, YouTube JS API, SHA512, Chrome Extensions

Trackfyre

A web-service which tracks specific DOM components of a website for changes. The front end interface was in the form of a Chrome Browser extension. Tracked components were handled by a MongoDB server in which query time could be specified along with method of notification.

Built with:

JavaScript, CSS, HTML, Chrome Extension, Python, MongoDB

CarAccel

An Android application which uses the built in accelerometer to calculate and display horizontal and lateral accelerations. It is integrated with the Google Maps API to create route tracking and plotting, with features to save route data and various other technical data. Won 1st place at Hack+@CMU.

Built with:

Android, Google Maps API

ChemX

A chemistry library on the Texas Instruments calculator architecture with a key-press GUI. It is optimized for program size, manages memory leaks, and handles program structure to work in a limited RAM environment. Over 1200 downloads.

Built with:

TI-Basic, BasicBuilder

Research

Pose Estimation in Symbiotic Rovers

Investigating position estimation and path-planning in multi-rover systems with limited sensing equipment.

DNA Read Mappers

SAFARI Research Group, Fall 2014

Building fast DNA Read Mappers for efficient DNA sequence analysis and target genome mapping. Designing novel algorithms and implementing new infrastructure.

CubeRover

16-861 Mobile Robot Development, Fall 2014

A semi-autonomous rover for commercial moon missions which fits in a 30cm cube. Working with a team which is designing and implementing the rover from scratch. Part of the software development team which is responsible for hazard detection, path planning, localization, and perception of the rover.

Predicting Stock prices using News Headlines

10-701 Machine Learning, Fall 2014

Researched predicting recent stock prices with Wall Street Journal and Dow Jones news articles for S&P 500 stocks. Used Markov random Fields with logistic regression and MCMC inference.

BitShred / BAP

Experimented with normalizations in feature hashing for improving efficiency in BitShred , a malware classification and semantic analysis tool. Integrated CMU's Binary Analysis Platform (BAP) to work with IDA, a high level disassembler.

Browsing Sensors

CMU Cylab Usable Privacy and Security Lab, Spring 2014

Developed browser sensing and data gathering technologies for the Security Behavior Observatory project, to be used in research studies. Added additional features to the Chrome and Firefox extensions and improved the workflow of the server-side retrievers.
