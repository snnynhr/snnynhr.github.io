About Me

I am currently a junior in Computer Science at Carnegie Mellon University. My interests lie in Algorithms, Data Structures, Machine Learning, Computer Security, and Distributed Cloud Systems.

Projects

MuSc

Engineered a web-based application which is able to split media across multiple devices, so that multiple laptops or phones could be used as a combined display.

ListenUP

Created an application that listened when someone called your name and subsequently lowered the volume of your computer and sent a notification to alert you, so that you could seamlessly listen to music and be aware of your surroundings. Built using Google's Speech to Text API. Won "Best Hack People Will Use Everyday".

ChromeTools

Developed Chrome extensions using HTML, CSS, and JavaScript. ChromeTools provides quick access to essential Chrome settings, and provides an interface for customized control of the browser's stored data.

YouTubeTracker

YouTubeTracker is a tool that manages YouTube history and displays recently watched videos, and additionally has the feature to convert and download the video as an MP3 file.

Trackfyre

Created an web-service which tracks specific DOM components of a website for changes. The front end interface was in the form of a Chrome Browser extension. Tracked components were handled by a MongoDB server in which query time could be specified along with method of notification.

CarAccel

Part of a team which developed an Android application which uses the built in accelerometer to calculate and display horizontal and lateral accelerations. It is integrated with the Google Maps API to create route tracking and plotting, with features to save route data and various other technical data. Won 1st place.

ChemX

Implemented and designed chemistry libraries on the Texas Instruments calculator architecture, with a key-press GUI while using efficient techniques to optimize program size, manage memory leaks, and handle program structure to work in a limited RAM environment.
